Скрипт на Питоне get_pyrus_json.py

Описание

Это простой скрипт на Питоне, который загружает задачи, формы и справочники из Пайруса при помощи Pyrus API.

Поддерживаются два формата вывода:
* JSON
* Человеческий (JSON отформатированный для печати при помощи Питон библиотеки pprint).

Оба варианта сохраняются в файлы, формата *.json и в простом текстовом. Далее можно использовать любимый просмотрщик
JSON для анализа. Cам автор скрипта для этих целей использует Chrome с расширением JSON Formatter. (Только не забудьте
включить настройку для отображения локальных файлов).

Использование скрипта:

* Внесите в файл credentials.json ваш логин и API ключ для Pyrus API (вы можете указать другой файл с логином/ключом
при помощи параметра -l)
* Запустите скрипт из директории, в которой он находится:
    * Чтобы получить данные задачи #1234567 в файлах t1234567.txt and t1234567.json
        $ python get_pyrus_json.py -t 1234567
      Если дополнительно указать ключ `-v 3`, то будет использована версия Pyrus API v3

    * Чтобы получить данные формы #123456  (будут созданы файлы f123456.txt и f123456.json соответственно).
        $ python get_pyrus_json.py -f 123456

    * Чтобы запустить скрипт с указанием другого файла с учетными данными используйте ключ -l
        $ python get_pyrus_json.py -f 123456 -l my_login.json

    * Чтобы открыть загруженный .json в браузере добавьте опцию -o :
        $ python get_pyrus_json.py -f 123456 -l my_login.json -o

Зависимости

Этот скрипт использует библиотеку pyrus-api - официальную Python библиотеку от Pyrus. Для установки просто наберите:
    pip install pyrus-api

Хорошей работы!
